## 1. 사용 도구
 
| 도구 | 용도 |
|------|------|
| Claude (claude.ai) | 기획, 설계, 코드 생성, 문서 작성 |
| Gemini API | 앱 내 AI 응답 (감정 공감 + 조언) |
| GitHub | 형상관리, 문서 버전 관리 |
 
---
 
## 2. AI Agent 활용 방식
 
### 기획 단계
- 주제 선정, 기능 범위 결정을 Claude와 대화하며 결정
- 요구사항 문서 초안을 Claude로 생성 후 직접 검토 및 수정
### 설계 단계
- 아키텍처, 디렉토리 구조 설계를 Claude에게 초안 요청
- 직접 이해하고 설명할 수 있는 수준으로 검토 후 확정
### 구현 단계
- 기능별 코드 생성을 Claude에게 요청
- 생성된 코드를 직접 읽고 동작 원리 파악 후 사용
