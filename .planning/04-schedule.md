# 04. 6주 일정
 
> 10주차 ~ 15주차
 
| 주 | 목표 | 산출물 | 검증 |
|----|------|--------|------|
| 10주차 | 기획·일정 | `.planning/*` | 본인 리뷰 |
| 11주차 | 설계·환경 | `docs/architecture.md`, 빌드 가능 상태 | "Hello World" 실행 |
| 12주차 | 핵심 기능 1 + **중간발표** | 동작하는 프로토타입 (입력 → AI 일정 생성 → 카드 UI) | 데모 가능 |
| 13주차 | 핵심 기능 2 + 테스트 | 지도 시각화, 히스토리 저장·조회 완성 | 테스트 통과 |
| 14주차 | 마감·배포·문서 | 배포 산출물, README, AUTHORING 문서 | 배포 링크 또는 빌드 |
| 15주차 | **최종 발표** | 발표 자료, 회고 | 평가 |
