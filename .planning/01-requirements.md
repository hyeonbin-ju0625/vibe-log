# 01. 요구사항
 
## 핵심 사용자 시나리오
 
### 시나리오 1 — 즉흥 여행러
> "주말에 갑자기 부산 가고 싶은데 뭐 해야 할지 모르겠어"
 
민준(25)은 친구들과 갑작스럽게 부산 여행을 결정했다.  
TR에 "부산, 2박 3일, 예산 30만원"을 입력하자  
AI가 동선이 최적화된 일정을 즉시 생성해줬다.  
지도에서 경로를 확인하고 바로 여행을 떠났다.
 
---
 
### 시나리오 2 — 계획 세우기 귀찮은 여행자
> "여행은 가고 싶은데 계획 짜는 게 너무 귀찮아"
 
지연(28)은 매번 여행 계획에 몇 시간을 쓰는 게 지쳐있었다.  
TR에 목적지와 기간만 입력했더니  
AI가 맛집, 관광지, 이동 동선까지 알아서 짜줬다.  
저장 기능으로 나중에 다시 꺼내볼 수도 있었다.
 
---
 
### 시나리오 3 — 여행 히스토리 관리자
> "저번에 갔던 여행 일정 다시 보고 싶은데"
 
현빈(23)은 TR로 만들었던 제주도 일정을 다시 참고하고 싶었다.  
히스토리 탭에서 날짜별로 저장된 일정을 바로 조회했다.  
이번엔 일정을 조금 수정해서 새 여행에 활용했다.
 
---
 
## 기능 요구사항 — MoSCoW 분류
 
| 분류 | 기능 |
|------|------|
| Must | 목적지·기간·예산 입력 |
| Must | Claude API로 AI 일정 자동 생성 |
| Must | 날짜별 일정 카드 UI 표시 |
| Must | Google Maps 장소 마커 시각화 |
| Must | 일정 로컬 저장 및 히스토리 조회 |
| Should | 일정 재생성 (마음에 안 들 때) |
| Could | 일정 텍스트 공유 기능 |
| Won't | 항공권·숙소 예약 연동 |
